Some Tips for Getting Started!
  • If you need to directly edit the html code on a section, click the "<> source" button on the text-editor menu.
    Be sure to click it again before you save any changes.
  • Applicant and contractor records are not editable in this panel, so that no personal information can be changed or deleted without a formal request by the individual to the website manager.
  • If you want sections added to the Home Page, please send a request to the website manager, and you will receive notification when the section is added and ready for content.
  • "CTRL(or CMD) + P" will print all applications and contractor contacts in resume form.
  • This site needs a record in the database for the home page, please make sure HOME.JSON is in the database pages collection.
    The page is editable afterwards, but must be initially installed by the website manager.
